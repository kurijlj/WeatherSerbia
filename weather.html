Weather Serbia is a web application designed to display current weather conditions for all major cities in Serbia. Application uses data published by Republic Hydrometeorological Service of Serbia on page http://www.hidmet.gov.rs/eng/osmotreni/index.rss.

Weather Serbia is hosted on Google App Engine platform and uses jQuery Mobile JavaScript library. Application code is written in Python 2.6.6 programming language and generates HTML5 and CSS3 code (due to use of GAE and jQuery Mobile code is not valid by means of w3 validation services).
